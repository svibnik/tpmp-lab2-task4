# Overview

Программа для работы с текстовыми файлами (вариант 10 — данные о покупателях).
Выполняет чтение данных из файла, обработку согласно заданию и запись результатов в новый файл.

# Author

Свибович Никита, группа 10

# Usage

1. Сборка: `make`
2. Запуск: `./bin/file_processor`
3. Входной файл: `data/input.txt`
4. Выходной файл: `data/output.txt`

# Additional Notes

Проект использует модель КИС для организации кода.
- Файлы с исходным кодом: `src/`
- Заголовочные файлы: `include/`
- Сборка автоматизирована с помощью Makefile и GitHub Actions
